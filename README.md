## About

**Random Chaos** is a server-side mod designed to bring unpredictability and fun to Hytale. Heavily inspired by the famous *GTA V Chaos Mod*, this plugin triggers a random event every **60 seconds**, completely changing gameplay rules, physics, and entity behaviors.

Whether you are running at Sonic speed, jumping with Moon Gravity, or fighting as a Tank, surviving is the only goal!

## Features

* Random Events: Over **10+ unique events** ranging from helpful buffs to deadly debuffs.
* Custom HUD: Real-time display of the chaos timer and active effects.

## üéÆ Commands

| Command | Description |
| :--- | :--- |
| `/chaos start` | Starts the chaos loop timer. |
| `/chaos stop` | Stops the loop and resets all active effects. |
| `/chaos events` | Lists all available events registered in the system. |
| `/chaos test <id>` | Forces a specific event to trigger immediately. |
| `/chaos add <player>` | Adds a specific player to the chaos game. |
| `/chaos remove <player>` | Removes a player from the chaos game. |
| `/chaos players` | Lists all players currently affected by chaos. |
